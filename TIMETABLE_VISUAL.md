## Interactions

### Adding a Task
1. **Double-click** on an empty cell
2. Fill in the "Add New Task" dialog:
   - Task Title
   - Task Type (dropdown)
   - Notes (optional)
3. Click **OK**
4. Task appears in the cell with appropriate color

### Viewing/Editing a Task
1. **Double-click** on a task cell
2. Opens appropriate dialog:
   - **Workout Details** for workout tasks
   - **Task Notes** for other task types
3. View or edit notes
4. Click **Save** to update

### Multiple Tasks in One Cell
1. **Single-click** on a cell with multiple tasks
2. View list of all tasks in that time slot
3. Click **View** on any task to open its dialog

## Implementation Details

### Data Binding
- Each cell dynamically queries `controller.model.getTasksForCell(day, hour, weekStart)`
- Tasks are filtered by week using ISO 8601 dateTime comparison
- Real-time updates when tasks are added/edited

### Color Coding
```javascript
function getTaskColor(taskType) {
    switch(taskType) {
        case "workout": return "#27ae60"  // Green
        case "work": return "#3498db"     // Blue
        case "meeting": return "#9b59b6"  // Purple
        case "personal": return "#e67e22" // Orange
        default: return "#95a5a6"         // Gray
    }
}
```

### Week Calculation
```javascript
function getWeekStart(date) {
    var d = new Date(date)
    var day = d.getDay()
    var diff = d.getDate() - day
    d.setDate(diff)
    d.setHours(0, 0, 0, 0)
    return d  // Returns Sunday of the week
}
```
